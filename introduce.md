本项目主要考虑的是船舶沿着一条测线下线之后，驶入相邻测线的路径优化问题；
已知，测线必定相互平行，测线之间间距固定，转弯半径R固定，船舶驶出测线之后需要走 Run out距离，
船舶进入另一条测线的时候，需要首先沿着测线平行方向行驶Run in 距离保证船舶后面的拖缆是拉直的平行于测线，从而使得船舶进入第二条测线的时候拖缆平行于测线方向。

在描述换线路径之前，首先明确几个关键几何概念：
- **EOL 数据点**：船舶在当前测线上采集结束的位置，即当前测线的 y_end 端点。
- **EOL 几何点**：船舶从 EOL 数据点沿测线航向继续行驶 Run out 距离后到达的位置，该点为换线路径的实际起点。船舶在此点的航向与测线航向一致。
- **SOL 数据点**：船舶在目标测线上采集开始的位置，即目标测线的 y_start 端点。
- **SOL 几何点**：船舶沿目标测线航向的反方向，从 SOL 数据点回退 Run in 距离后到达的位置，该点为换线路径的实际终点。船舶在此点的航向与目标测线航向一致。
- **转弯圆**：以转弯半径 R 为半径、圆心位于船舶侧方的圆。左转（L）时圆心在船舶航向的左侧，右转（R）时圆心在船舶航向的右侧。
- **公切线**：连接两个转弯圆的切线段。当两个转弯方向相同（如 LSL 或 RSR）时使用外公切线（切线不穿过两圆圆心连线），当两个转弯方向相反时使用内公切线（切线穿过两圆圆心连线之间）。

所以船舶从某一条测线行驶到第二条测线的路径，可分为三种情况：

（1）当测线间距 W 满足 W > 2R 时的 Dubins CSC 换线路径

当测线间距 W > 2R 时，两个转弯圆之间存在足够的空间，可以通过"圆弧—直线—圆弧"（CSC，即 Curve-Straight-Curve）的 Dubins 路径形式完成换线。此时仅考虑 LSL 与 RSR 两种路径类型：

- **LSL 路径**：船舶从 EOL 几何点出发，先左转（L）沿第一个转弯圆弧行驶至外公切线的切点 T1，然后沿外公切线直线行驶至第二个转弯圆上的切点 T2，最后再左转（L）沿第二个转弯圆弧行驶至 SOL 几何点。LSL 路径的两个转弯圆分别位于 EOL 几何点的左侧和 SOL 几何点的左侧，两圆半径均为 R，外公切线连接两圆同侧切点且不穿过两圆圆心连线。

- **RSR 路径**：与 LSL 对称，船舶先右转（R）沿第一个转弯圆弧行驶至切点 T1，然后沿外公切线直线行驶至切点 T2，最后再右转（R）沿第二个转弯圆弧行驶至 SOL 几何点。RSR 路径的两个转弯圆分别位于 EOL 几何点的右侧和 SOL 几何点的右侧。

算法对 LSL 和 RSR 两种路径均进行求解，并选择总路径长度最短者作为最优换线路径。由于 W > 2R，两个转弯圆的圆心距大于 2R，保证了外公切线一定存在且直线段长度为正，从而使 CSC 路径始终可行。

该场景下完整的换线路径由以下航段依次组成：Run out 直线段（从 EOL 数据点沿测线方向行驶至 EOL 几何点）→ 第一段圆弧 → 公切线直线段 → 第二段圆弧 → Run in 直线段（从 SOL 几何点沿目标测线方向行驶至 SOL 数据点）。路径在每个连接点处均保证位置连续和航向连续。

（2）当测线间距 W 满足 W = 2R 时的半圆换线路径

当 W = 2R 时，两测线间距恰好等于船舶最小转弯直径，此时两个转弯圆刚好外切，不存在公切线直线段，因此无法使用常规的 CSC 路径。该场景采用半圆形转弯完成换线，其几何结构如下：

- **半圆圆心**：位于两测线之间连线的中点，即 ((x1 + x2) / 2, y_star)，其中 y_star 为半圆所在的 Y 坐标高度。
- **半圆起点**：位于当前测线（line1）上，坐标为 (x1, y_star)，船舶在此点开始半圆转弯。
- **半圆终点**：位于目标测线（line2）上，坐标为 (x2, y_star)，船舶在此点完成半圆转弯。
- **半圆弧长**：由于半圆半径为 R，弧长恒为 πR。
- **转弯方向**：根据半圆起点处的切线方向与船舶航向的一致性确定。在半圆起点处，沿半圆有两个切线方向（左切和右切），算法选择与船舶在 EOL 几何点的航向更一致（点积更大）的切线方向所对应的转弯方向（L 或 R）。

半圆高度 y_star 的确定：算法依次尝试以 EOL 几何点的 Y 坐标和 SOL 几何点的 Y 坐标作为 y_star 候选值，对每个候选值进行可行性检验：
1. 半圆起点必须在 EOL 几何点的航向前方（即船舶沿当前航向可以到达半圆起点）；
2. 半圆终点必须在 SOL 几何点的航向后方（即从半圆终点出发沿目标测线航向可以到达 SOL 几何点）。
通过检验的候选路径中，选择总长度最短者。

该场景下完整的换线路径由以下航段依次组成：Run out 直线段 → 前置直线段（从 EOL 几何点沿测线方向行驶至半圆起点，长度可能为零）→ 半圆弧 → 后置直线段（从半圆终点沿目标测线方向行驶至 SOL 几何点，长度可能为零）→ Run in 直线段。当 EOL 几何点恰好位于半圆起点时，前置直线段长度为零；当半圆终点恰好位于 SOL 几何点时，后置直线段长度为零。

（3）当测线间距 W 满足 W < 2R 时的固定对称三圆弧换线路径

当测线间距 W 满足 W<2R，其中 R 为船舶最小转弯半径时，船舶无法通过常规的两圆弧加直线段形式完成相邻测线之间的换线，此时将其归入小间距换线场景。对于该类场景，本文采用固定对称三圆弧换线模板进行路径构造。该模板由三段半径均为 R 的圆弧组成，两侧圆弧关于两测线中垂线对称，中间主圆弧的圆心位于两测线中点的垂线方向上，从而保证整条换线路径在几何上连续且满足船舶最小转弯半径约束。根据船舶换线方向不同，当船舶由左侧测线驶向右侧测线时，路径类型记为 LRL；当船舶由右侧测线驶向左侧测线时，路径类型记为 RLR。此外，为满足拖缆作业过程中对拖缆姿态的要求，船舶在驶出当前测线后需先沿测线方向继续航行 Run out 距离，在进入目标测线前则需预留 Run in 距离，使拖缆重新拉直并与目标测线方向保持平行。对于小间距换线情形，三圆弧模板的起转高度由当前测线出口几何点 EOL_geom 与目标测线入口几何点 SOL_geom 的相对位置共同确定：若船舶驶出 Run out 后尚未达到起转所需高度，则需先补充一段沿测线方向的直线航段；若三圆弧完成后尚未到达目标测线入口几何点，则需在目标测线延长线上补充一段直线航段后再进入 Run in。因此，该场景下的换线路径由"前置直线段 + 对称三圆弧模板 + 后置直线段"构成，从而在满足转弯半径约束的同时，实现船舶及拖缆系统在相邻平行测线之间的安全、连续与可作业换线。
