本项目主要考虑的是船舶沿着一条测线下线之后，驶入相邻测线的路径优化问题；
已知，测线必定相互平行，测线之间间距固定，转弯半径R固定，船舶驶出测线之后需要走 Run out距离，
船舶进入另一条测线的时候，需要首先沿着测线平行方向行驶Run in 距离保证船舶后面的拖缆是拉直的平行于测线，从而使得船舶进入第二条测线的时候拖缆平行于测线方向。

故分为三种情况：
（1） 对于一般工程中的相邻平行测线换线问题，当测线间距 W 满足 W>2R ，其中 R 为船舶最小转弯半径时，可将其归入“大间距、同向换线”场景。对于该类场景，本文采用仅考虑 LSL 与 RSR 的换线路径规划算法进行求解。
（2） 当测线间距 W 满足 W = 2R，其中 R 为船舶最小转弯半径时，船舶通过拐弯半圆换线到另一条测线方向上。
（3） 当测线间距 W 满足 W<2R，其中 R 为船舶最小转弯半径时，船舶无法通过常规的两圆弧加直线段形式完成相邻测线之间的换线，此时将其归入小间距换线场景。对于该类场景，本文采用固定对称三圆弧换线模板进行路径构造。该模板由三段半径均为 R 的圆弧组成，两侧圆弧关于两测线中垂线对称，中间主圆弧的圆心位于两测线中点的垂线方向上，从而保证整条换线路径在几何上连续且满足船舶最小转弯半径约束。根据船舶换线方向不同，当船舶由左侧测线驶向右侧测线时，路径类型记为 LRL；当船舶由右侧测线驶向左侧测线时，路径类型记为 RLR。此外，为满足拖缆作业过程中对拖缆姿态的要求，船舶在驶出当前测线后需先沿测线方向继续航行 Run out 距离，在进入目标测线前则需预留 Run in 距离，使拖缆重新拉直并与目标测线方向保持平行。对于小间距换线情形，三圆弧模板的起转高度由当前测线出口几何点 EOL_geom 与目标测线入口几何点 SOL_geom 的相对位置共同确定：若船舶驶出 Run out 后尚未达到起转所需高度，则需先补充一段沿测线方向的直线航段；若三圆弧完成后尚未到达目标测线入口几何点，则需在目标测线延长线上补充一段直线航段后再进入 Run in。因此，该场景下的换线路径由“前置直线段 + 对称三圆弧模板 + 后置直线段”构成，从而在满足转弯半径约束的同时，实现船舶及拖缆系统在相邻平行测线之间的安全、连续与可作业换线。
